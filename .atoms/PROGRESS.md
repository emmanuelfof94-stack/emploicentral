# Requirements & Progress

## Requirements Overview
Build an intelligent job platform that centralizes job offers, analyzes user CVs with AI, calculates compatibility scores, and sends alerts for relevant jobs.

## User Stories
- As a user, I can browse centralized job offers with filtering
- As a user, I can upload my CV and have it analyzed by AI
- As a user, I can see compatibility scores between my profile and job offers
- As a user, I can set alert preferences for relevant jobs
