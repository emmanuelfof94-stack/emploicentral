# Project Summary
JobMatch AI is an intelligent job platform designed to enhance the job-seeking experience by centralizing job offers from various sources. It leverages AI to analyze user CVs, calculate compatibility scores, and provide real-time alerts for relevant job opportunities. The platform is fully translated into French and aims to be accessible to a diverse audience, particularly young people in West Africa, with a focus on job opportunities in Côte d'Ivoire and surrounding regions.

# Project Module Description
The project consists of several functional modules:
- **Landing Page**: Marketing page introducing the platform's features, now emphasizing job demand statistics in Côte d'Ivoire, and providing free access for West African youth.
- **Authentication**: Manages user login and registration, integrating with Atoms Cloud for secure authentication.
- **Dashboard**: Displays user-specific information, job matches, and profile completion status. Includes a manual trigger for compatibility score calculations.
- **Jobs**: Lists job offers with advanced search and filtering capabilities, focused solely on West African job opportunities.
- **Profile**: Manages user profiles and CV uploads, with recent fixes for saving profile information.
- **Alerts**: Configures user preferences for job alerts.
- **Backend API**: Provides endpoints for CV analysis and compatibility scoring.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: FastAPI, PostgreSQL, Object Storage, AI Hub (Atoms Cloud)
- **AI Tools**: analyzepdf for CV extraction, deepseek-v4-pro for compatibility scoring

# Usage
To set up the project, follow these steps:
1. **Install dependencies**:
   - For the backend: `pip install -r app/backend/requirements.txt`
   - For the frontend: `pnpm install`
   
2. **Build the project**:
   - For the backend: Run the application using `uvicorn app.backend.main:app --reload`
   - For the frontend: `pnpm run build`

3. **Run the application**:
   - Start the backend server.
   - Start the frontend server.
